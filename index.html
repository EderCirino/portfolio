InviteGuest Web App

Descrição:

Um webapp para controlar festas, eventos, etc. Onde é possível registrar os convidados pelo nome.

Desenvolvido com:
  • HTML5
  • CSS3
  • javascript(ES6)
Conclusão:

Usando um pouco de propriedades CSS3 foi possível fazer uma página funcional. Sobre Javascript, todos os dados digitados no aplicativo são armazenados no LocalStorage(tipo um banco de dadaos, porém no browser) e foi necessário usar arrays, função filter. O poder do Javascript no DOM é formidável e é possível gerenciar o DOM com eficiência.
